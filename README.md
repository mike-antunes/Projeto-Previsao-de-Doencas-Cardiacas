# Projeto de previsão de doenças cardíacas com Machine Learning

## Resumo do projeto:
O objetivo do projeto é utlizar a base de dados que contém as informações sobre problemas cardíacos de pacientes contendo informações importantes para definir quem
tem possíveis problemas cardíacos e quem não tem possíveis problemas cardíacos.  
Com base nisso, utilizamos o Dataset para fazer todo um processo de tratamento em ciência de dados para podermos ter a capacidade de criar um modelo de Machine Learning para prever quais pacientes tem possíveis problemas cardíacos ou não.

## Observação:
Utilizamos **Random Forest** para o nosso modelo de previsão. Foi feito a análise, tratamento, normalização, análise de possíveis outliers, balanceamento dos dados, utilização da métrica de **classification_report** e **confusion_metrics**, verificação a accuracy, entre outras avaliações.
